align-self

align-self: flex-start - розміщується на початку контейнера по осі Y (1)

align-self: flex-end - розміщується в кінці контейнера по осі Y (2)

align-self: center - розміщується посередині контейнера по осі Y (3)

align-self: stretch - заповнює по осі Y всю довжину контейнера (4)

justify

justify-content, justify-items, justify-self - розміщення елементів в контейнері, працює так, як align-content, align-items та align-self, але по осі Х, горизонтально

place

place-content, place-items, place-self - поєднує в собі значення align-& та justify-&

&-content - використовується для розподілу вільного місця між блоками

&-items - &-self, але для всіх елементів в контейнері

&-self - вирівнювання елемента згідно значенню по вільному місцю, якщо інші елементи займають якесь місце, то вирівнювання буде по вільному місцю, а не по загальній довжині/ширині контейнера

CSS Flexbox

display

display: flex - застосовується для контейнера flex-елменетів, всі вкладені елементи стають flex-елементами та всі блоки розміщуються в ряд. контейнер займає всю ширину екрану. якщо не задавати розміри flex-елементам, то ширина буде підлаштовуватися під контент, а висота займати 100% flex-контейнера
display: inline-flex - все те саме, але ширина контейнеру підлаштовується під вміст
