accent-color

accent-color: #000000 - змінює колір чекбоксу, прогресу, радіокнопки або діапазону

animation-name

animation-name: name - назва анімації, щоб зв'язати з @keyframes

@keyframes

@keyframes name {
	0% {}
	100% {}
}
встановлює CSS властивості на початку, кінці та проміжні при необхідності, які будуть змінюватися в ході анімації

animation-duration

animation-duration: 3s - тривалість анімації від 0% до 100%; ці 3 значення є обов'язковими для створення анімацій

animation-timing-function

animation-timing-function: ease - перехід починається повільно, швидко розганяється і сповільнюється в кінці (встановлено за замовчуванням)
animation-timing-function: ease-in - перехід починається повільно, а потім плавно прискорюється в кінці

Результат:


animation-timing-function: ease-out - перехід починається швидко і плавно сповільнюється в кінці

Результат:


animation-timing-function: ease-in-out - перехід повільно починається і повільно закінчується

Результат:


animation-timing-function: linear - перехід відбувається рівномірно протягом усього часу, без вагань у швидкості

Результат:

animation-iteration-count

animation-iteration-count: 4 - задає к-сть разів повторення анімації

Результат:


animation-iteration-count: infinite - нескінченно повторює анімацію

animation-direction

animation-direction: reverse - кадри програватимуться у зворотному порядку

animation-direction: alternate-reverse - кожен непарний прохід по кадрах програватиметься у зворотному порядку

animation-play-state

animation-play-state: paused - зупиняє анімацію при події, в даному випадку використовується наведення на коло

animation-delay

animation-delay: 3s - затримує початок анімації на 3 секунди

Результат:

animation-fill-mode

animation-fill-mode: forwards - після закінчення анімації примінює властивість, яка знаходиться на 100%

Результат:

animation

animation: |animation-name|, |animation-duration|, |animation-timing-function|, |animation-iteration-count|, |animation-direction|, |animation-delay|, |animation-fill-mode| - використовується дли швидкого запису значень тегів, всі значення потрібних властивостей записуються відповідно вказаного порядку
